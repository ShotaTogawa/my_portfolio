# Book Log  
***
##概要  
所持している本の進捗度を管理するサービスです。それぞれの本に対してメモを残せます。

##機能  
[x] ログイン、サインアップ、ログアウト
[x] 本の登録機能/編集/削除
[x] 本のステータス管理
[x] 本の一覧機能
[x] 本の詳細
[x] メモ、登録・削除
[x] ステータスごとの冊数確認機能
[x] 所持している本のジャンル確認機能(円グラフ,テーブル)
[] ページネーション
[] ソート

##使用技術
- React
- Redux
- Semantic UI
- BootStrap
- Nodejs(Express)
- MongoDB(mangoose)
- Jest(未完)
